## 🎬 Особенности Дизайна:

### Персонаж Cowgirl:
- 🤠 Ковбойская шляпа (покачивается)
- 👩‍🦰 Лицо дилера
- 👈👉 Руки указывают на табличку
- **Плавающая анимация** (вверх-вниз 3s)

### Деревянная Табличка:
- 🎨 **Цвет**: Коричневый градиент (#8b4513 → #654321)
- 🪵 **Текстура**: Линии имитируют дерево
- 🪢 **Декор**: Верёвки по углам
- 🏆 **Заголовок**: "WINNER!" (золотой, светящийся)

### Информация О Победе:
- **Имя победителя**: Большой коричневый текст (Georgia serif)
- **Комбинация**: Красный цвет для названия руки
- **Сумма выигрыша**: Огромный зелёный текст ($800)
- **Футер**: Иконки казино 🎰 ★ 🃏

### Анимации:
- ✨ **Banner появляется**: Вращение + масштаб (0.6s)
- 🎩 **Шляпа качается**: Влево-вправо
- 👈👉 **Руки указывают**: Движутся к табличке
- 💰 **Сумма пульсирует**: Scale pulse (1s)
- 🏆 **Трофеи прыгают**: Bounce (1s)
- ⭐ **Sparkles для больших выигрышей** (>$500): Летят вверх

## 📊 Технические Детали:

### CSS Анимации (9 штук):
1. `bannerAppear` - Появление баннера (вращение + scale)
2. `float` - Плавание персонажа (up/down)
3. `hatTip` - Покачивание шляпы
4. `pointLeft/pointRight` - Движение рук
5. `glowPulse` - Пульсация свечения заголовка
6. `bounce` - Прыжки трофеев
7. `scalePulse` - Пульсация суммы
8. `sparkleFloat` - Полёт звёздочек

### Деревянная Текстура:
```css
repeating-linear-gradient(90deg, ...) /* Горизонтальные линии */
repeating-linear-gradient(0deg, ...)  /* Вертикальные линии */
```

### Цветовая Палитра:
- 🟤 Дерево: `#8b4513`, `#654321`, `#d2691e`
- 🟡 Золото: `#ffd700`, `#ffa500`
- 🔴 Акцент руки: `#dc2626`
- 🟢 Деньги: `#16a34a`
- 🟠 Бежевая карта: `#fff9f0`, `#ffefd5`

## 🔥 Основные Фичи:

### 1. Emoji-Based Персонаж
**Преимущества:**
- ✅ Не нужны внешние изображения
- ✅ Работает сразу без загрузки
- ✅ Легко модифицировать
- ✅ Совместимость со всеми браузерами

**Можно заменить на настоящее изображение:**
```tsx
// В будущем можно заменить на:
<img src="/assets/poker-dealer.png" alt="Dealer" />
```

### 2. Responsive Design
Адаптируется под мобильные:
```css
@media (max-width: 600px) {
  /* Уменьшенные размеры текста */
  /* Компактный персонаж */
  /* Меньший padding */
}
```

### 3. Sparkles для Больших Выигрышей
```tsx
{potAmount >= 500 && (
  <Sparkles /> // Показываем только для $500+
)}
```
